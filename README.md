# Plague Call Online Character List

Это электронный лист персонажа.

Создан по подобию физического листа персонажа по системе "Чумной зов", которая была создана Коластом.

Группа VK:
[https://vk.com/trpg_call](https://vk.com/trpg_call)

## Что это

Это статическое веб-приложение с листом персонажа:

- поля заполняются прямо в браузере;
- точки можно отмечать кликом;
- данные сохраняются локально в браузере;
- есть светлая и тёмная тема.

## Файлы проекта

- `index.html` - структура страницы;
- `styles.css` - оформление и темы;
- `script.js` - интерактивность, сохранение и логика точек.
